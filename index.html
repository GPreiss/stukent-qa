2. What language is the following code in and what does it do?

def get_it(*nums)
num = nums.first
nums.each do |n|
num = n if n > num
end
num
end

Cool! This is in Ruby (the cleanest and prettiest language- slight bias here).
From what I understand, the goal of this method is to get the highest value from an array. Nums is the array. Num is the first element of that array. Then the each loop goes through each element of the array, searching for the highest value. The first element it sees is initally the highest(num), then compares each element after (n), to see if it's higher than the first, and if it is, that's the return value.
